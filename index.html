Instalación Windows 11 (Bypass)

1. Saltar TPM 2.0 / CPU

En el USB (carpeta sources), abre CMD y escribe:
setupprep.exe /product server

2. Cuenta Local (OOBE)

Al pedir cuenta, presiona Shift + F10 y escribe:
start ms-cxh:localonly

Opción B: Bypass Internet (NRO)

Si el anterior no funciona, usa este para reiniciar sin requisitos de red:
OOBE\BYPASSNRO
