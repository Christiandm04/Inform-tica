Instalación Windows 11 (Bypass)

1. Saltar Requisitos (No TPM 2.0 / CPU)

Dentro del USB de instalación, ve a la carpeta sources, abre CMD y escribe:
setupprep.exe /product server

* Esto permite actualizar desde Windows 10 o instalar limpio saltando el check de hardware.

2. Bypass Cuenta Microsoft (OOBE)

Cuando pida internet/cuenta, pulsa Shift + F10 y escribe:
start ms-cxh:localonly

* Te llevará directamente a la creación de la cuenta local.
