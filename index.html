Bypass Windows 11 (No TPM)

Paso 1: USB

Tener en el USB la ISO de W11, entrar a la carpeta sources y abrir CMD.

Paso 2: Comando Bypass
setupprep.exe /product server
Paso 3: Salto de Cuenta

En la pantalla de cuenta Microsoft, pulsa Shift + F10 y ejecuta:

start ms-cxh:localonly

Esto te permite crear la cuenta local sin internet.
